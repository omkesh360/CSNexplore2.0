# Quick Start: Regenerate All Listing Pages

## In 3 Steps

### Step 1: Login to Admin
- Go to your admin dashboard
- Login with admin credentials

### Step 2: Click Regenerate
- In the sidebar, click **"Regenerate"**
- Or go to `/admin/regenerate-pages.php`

### Step 3: Click the Button
- Click **"Regenerate Now"**
- Wait for completion (usually 5-30 seconds)
- See the results

## What Happens

✓ All listings get the default Aurangabad map
✓ Similar listings feature is enabled
✓ Existing custom maps are preserved
✓ All pages are ready to display

## After Regeneration

### View Results
- Check the summary showing how many listings were updated
- See breakdown by category

### Customize Maps (Optional)
1. Go to Admin → Map Embeds
2. Select a category
3. Find a listing
4. Click "Edit Map"
5. Paste a custom Google Maps embed code
6. Save

## That's It!

Your listing pages now have:
- 📍 Embedded Google Maps
- 🔄 4 Random similar listings
- 📱 Mobile-responsive design
- ⚡ Fast loading

## Need Help?

**Maps not showing?**
- Check if regeneration completed successfully
- Verify the map embed code is valid

**Want custom maps?**
- Use the Map Embeds admin panel
- Get codes from Google Maps → Share → Embed

**Something broken?**
- Check `/logs/php_errors.log`
- Verify database connection
- Test from a different browser
