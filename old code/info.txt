================================================================================
  PROJECT OVERVIEW
================================================================================
CSNExplore is a city-specific travel discovery and booking platform for
Chhatrapati Sambhajinagar (formerly Aurangabad), Maharashtra, India.
It lets users browse 6 service categories (stays, cars, bikes, restaurants,
attractions, buses), read 80+ local travel blogs, and submit booking requests.
Admins manage all content, listings, bookings, users, and site settings via
a full-featured admin panel.

AUTH API (/api/auth/)
  POST /api/auth/login          - Login, returns JWT token + user object
